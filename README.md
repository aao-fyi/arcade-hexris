# Hexris
An addictive puzzle game inspired by Tetris.

This project is a fork of [Hextris](https://github.com/Hextris/hextris). This version includes some interface changes and has been restructured for Hugo site generation.

+ [arcade.aao.fyi/hexris](https://arcade.aao.fyi/hexris)
